# 图片目录

请将您的图片文件放在此目录中：

- `profile.jpg` - 个人头像照片（建议尺寸：400x400 像素）
- `image1.jpg` - 第一张展示图片
- `image2.jpg` - 第二张展示图片
- `image3.jpg` - 第三张展示图片

## 支持的图片格式
- JPG/JPEG
- PNG
- GIF
- WebP

## 图片建议尺寸
- 头像：400x400 像素（圆形显示）
- 展示图片：建议宽度 600-800 像素，保持合适的宽高比

## 使用说明
图片添加到此目录后，README.md 会自动引用这些图片。确保图片文件名与 README.md 中的引用名称一致。
